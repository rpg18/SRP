The Gene Database

This page allows the querying of PigeonPigs gene counts matrix.

Simply enter a gene name in the query box below, and a table of every cell showing expression of that gene will be returned, along with useful information such the cell tissue type, absolute read count, cell type assigned by Darmanis et al. (2015) and the PigeonPigs cluster assignment of that cell!
